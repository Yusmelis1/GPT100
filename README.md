# GPT-100 Pro

Sistema profesional de análisis y pronóstico de fútbol basado en el método GPT-100.

## Objetivo

Automatizar:

- Escaneo mundial de partidos oficiales.
- Aplicación del método GPT-100.
- Clasificación de partidos:
  - ⭐ Nivel S
  - 🟢 Nivel A
  - 🟡 Nivel B
  - 🔴 Nivel C
- Auditoría automática.
- Generación de informes PDF.
- Historial en Excel.

## Método GPT-100

### 1. Forma reciente (25%)

- Últimos 4-5 partidos.
- Victorias, empates y derrotas.
- Racha.

### 2. Ataque (20%)

- Goles.
- Promedio.
- Calidad ofensiva.

### 3. Defensa (25%)

- Goles recibidos.
- Porterías a cero.
- Solidez.

### 4. Dificultad de rivales (20%)

- Valoración de rivales.
- Dificultad media.

### 5. Otros factores (10%)

- Localía.
- Lesiones.
- Sanciones.
- Posesión.
- Córners.
- Historial útil.
